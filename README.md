- [nodejs](https://nodejs.org/en/)

## Installation

Clone the project

```bash
  git clone https://github.com/renatoallart/mealsApplication.git
```

Go to the project directory

```bash
  cd mealsApplication
```

Install dependencies

```bash
  npm install
  #or
  yarn add
```

Start the application

```bash
  npm run dev
  #or
  yarn dev
```
